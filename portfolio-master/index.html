About Me

My name is Alex Sherrer and I am a web and UI designer. I am very passionate about building websites and UIs to create the best possible design for the website. I have been passionate in design since a very young age. My main hobbies outside of development include gaming, sports, music, and designing posters. I have been working on four different projects. I have created an online tech store using Wordpress, Elementor Pro, and WooCommerce. I have also created two different wordpress themes using HTML, SASS preprocessor, PHP, and JavaScript. I have also created a weather app that uses an API. It can display the temperature, forecast, and the HI/LOW temperature. All of my projects use responsive design. Some other skills I have include Adobe Photoshop, Illustrator, XD, and Premiere Pro.
